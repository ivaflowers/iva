l'objet

About the project

My task was to design an e-commerce website. Idea was to showcase a high end product in a minimalistic and premium approach in order to help customers effortlessly browse and purchase jewellery.

Challenge was to come up with an online jewlery shop that is not confusing for users to explore and help achieve their goals with ease, while simultaneously offering enjoyable visual experience.


Solution

Home screen is very straightforward and consists of bold, vertically aligned images and names ot the collections. Both images and text are clickable making it easy and intuitive to user to reach the product pages. Additional options are in the hamburger menu which has a fixed position when scrolling being available for the user at all times.

Product page allows user to see more detailed information about pieces and select options that they prefer (material, size). Drop down menus are designed to be discreet but easy to spot and interact. When product is added to the bag, layover offering user to go to the bag or continue shopping is popping up.

Shopping process is created to allow user to buy with or without creating an account. Clear info and linear process are created in order to give simple and no-brainer experience without complicating it too much. All the steps (personal, shipping and payment details are numerated for user to know in which part of the process they are.
